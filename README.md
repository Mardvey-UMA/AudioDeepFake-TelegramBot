# AudioDeepFake-TelegramBot
 Telegram Bot for detect DeepFake audio
Проект телеграмм бота разработанный на языке программирования python
Бот предназначен для выявления DeepFake аудиозаписей 
Для выявления была обучена нейронная сеть, её веса находятся в папке model
Для того, чтобы бот смог проанализировать вашу аудиозапись необходимо отправить её боту
Далее вы получите ответ: является ли аудиозаписей подделкой или нет
##############################################################################
Bot telegram project developed in python programming language
The bot is designed to detect DeepFake audio recordings
For detection, a neural network was trained, its weights are in the model folder
In order for the bot to be able to analyze your audio recording, you must send it to the bot
Next, you will receive an answer: is the audio recording fake or not
